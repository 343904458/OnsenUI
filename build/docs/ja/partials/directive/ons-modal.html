ons-modal

画面全体をマスクするモーダル用コンポーネントです。下側にあるコンポーネントは、 モーダルが表示されている間はイベント通知が行われません。

属性 #

名前 型 / デフォルト値 概要

メソッド概要 #

シグネチャ 概要
toggle() モーダルの表示を切り替えます。
show() モーダルを表示します。
hide() モーダルを非表示にします。
getDeviceBackButtonHandler() ons-modalに紐付いているバックボタンハンドラを取得します。

メソッド

toggle() #

モーダルの表示を切り替えます。

show() #

モーダルを表示します。

hide() #

モーダルを非表示にします。

getDeviceBackButtonHandler(): Object #

ons-modalに紐付いているバックボタンハンドラを取得します。

返り値: デバイスのバックボタンハンドラを返します。
