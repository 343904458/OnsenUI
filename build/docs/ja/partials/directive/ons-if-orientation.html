ons-if-orientation

画面の向きに応じてコンテンツの制御を行います。portraitもしくはlandscapeを指定できます。すべての要素の属性に使用できます。

  • 使い方
  • 属性
  • 関連情報
  • 議論

使い方 # 

<div ons-if-orientation="portrait">
  <p>This will only be visible in portrait mode.</p>
</div>

関連情報 #

  • 他のユーティリティAPI
  • ons-if-platformコンポーネント

属性 #

名前 型 / デフォルト値 概要
ons-if-orientation String portraitもしくはlandscapeを指定します。 Optional
