ons-col

グリッドシステムにて列を定義します。ons-rowとともに使用し、コンポーネントのレイアウトに利用します。

  • 使い方
  • 実例
  • 属性
  • 関連情報
  • 議論

使い方 # 

<ons-row>
  <ons-col width="50px"><ons-icon icon="fa-twitter"></ons-icon></ons-col>
  <ons-col>Text</ons-col>
</ons-row>

注意: Android 4.3以前、もしくはiOS 6以前のOSの場合、ons-rowとons-columnを組み合わせた場合に描画が崩れる場合があります。

属性 #

名前 型 / デフォルト値 概要
align String 縦の配置を指定する。"top", "center", "bottom"のいずれかを指定します。 Optional
width String カラムの横幅を指定する。パーセントもしくはピクセルで指定します（10%や50px）。 Optional
